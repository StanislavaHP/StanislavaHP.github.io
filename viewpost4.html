Lecture Notes 4: Arrays C#


  1. Definition : Collection of variables called elements. The elements are numbered, the numbers are called indices. The elements of a given array are of the same type. One dimensional arrays are called vectors, two dimensional arrays are called matrices.
  2. Declaring an Array
    1. DataType[] arrayName
      At this point the declared array is a reference which does not have a value – it points to null. The memory for the elements is not yet allocated.
    2. Allocate memory to the array with new
      1. DataType[] arrayName = new DataType[x];
      2. x – x elements are allocated in the dynamic memory
      3. The default initial value for all elements is 0 for numeral types, null for reference types and false for the bool type.
    3. Allocate memory to the array explicitly
      1. DataType[] arrayName = {1,2,3,4,5,6};
      2. In this way we create and initialize the elements of the array at the time of the declaration. We use {} instead of new.
    4. Once memory is allocated an array has fixed length.
  3. Access to the elements of an array
    1. arrayName[index] = x; (the element at that index would be assigned the value of x)
    2. Iteration through the elements of an array
      1. For – allows both changes and access to the elements of an array
      2. Foreach – allows only access to the elements of an array
  4. Split (x, StringSplitOptions.RemoveEmptyEntries) – allows you to split a string into an array.
    1. Split the string by x. x can be a string, char or array of strings or chars.
    2. StringSplitOptions.RemoveEmptyEntries – optional, used when you split by multiple chars or strings. For example if you are splitting by comma and space and have consecutive comma and space within the string you are splitting, your array would contain empty entries, which using this string split option will remove.
  5. Linq
    1. arrayName.Select(int.Parse).ToArray() – allows you to iterate through each element within the array and convert a string into an int. Linq returns a collection of type IEnumerable. ToArray() converts the collection of type IEnumerable into an array.
    2. Array.Reverse(arrayName); - reverse the array
    3. Array.Copy(sourceArrayName, destinationArrayName, how many elements we need to copy); - destination Array cannot be null – it does not have to be initialised but memory should have been allocated for it
    4. Array.IndexOf (arrayName, element)
  6. Multidimensional arrays (rows and columns)
    1. Declaration: Int[,] ArrayName – there is a comma for each dimension, excluding the first one
    2. Memory Allocation: Int[,] ArrayName = new int[x,y] – we set the size of each dimension (x rows, y columns). The number of elements = x*y.
    3. Initialization: Int[,] ArrayName = { {x,y,z},{x2,y2,z2} }
    4. Accessing: ArrayName[x,y]. Each element is unique and can be identified with indices from the array
    5. Length: Each dimension of an array has its own length and its own index. ArrayName.GetLength(0) – gives you the length of the first dimension
    6. Iteration through the element of an array: Nested for loops – print, read, populate
  7. Arrays of arrays (Jagged arrays)
    1. Definition: Arrays in which each row contains an array of its own, and that array can have length different than those in the other rows.
    2. Declaration, Allocation and Initialisation:
      1. Declare: int[][] ArrayName;
      2. Allocate:
        1. ArrayName = new int[2][]
        2. ArrayName[0] = new int[5];
        3. ArrayName[1] = new int[3];
      3. Initialise: Int[][] arrayName = {new int[] {x,y,z}, new int[] {x2,y2,z2}, new int[] {x3,y3}}
      4. The elements of a jagged array can be one – dimensional and multi – dimensional arrays
        1. Int [] [,] ArrayName = new int [x][,];
        2. ArrayName[0] = new int[,] {{x1,x2},{y1,y2}};
        3. ArrayName[1] = new int[,] {{z1,z1},{j1,j2}};
    3. Accessing elements: ArrayName[x][y]
