Lecture Notes 4: Arrays C#

  1. Definition : Collection of variables called elements. The elements are numbered, the numbers are called indices. The elements of a given array are of the same type. One dimensional arrays are called vectors, two dimensional arrays are called matrices.
  2. Declaring an Array
    1. DataType[] arrayName (at this point the declared array is a reference which does not have a value – it points to null. The memory for the elements is not yet allocated)
    2. Allocate memory to the array with new - DataType[] arrayName = new DataType[x]
      1. x – x elements are allocated in the dynamic memory
      2. The default initial value for all elements is 0 for numeral types, null for reference types and false for the bool type.
    3. Allocate memory to the array explicitly
      1. DataType[]arrayName = {1,2,3,4,5,6};
      2. In this way we create and initialize the elements of the array at the time of the declaration. We use {} instead of new.
    4. Once memory is allocated an array has fixed length.
